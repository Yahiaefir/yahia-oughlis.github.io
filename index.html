Sétif, Algérie (À distance)

Déroulement du stage – Les 5 étapes

J'ai réalisé ce stage à distance pour une agence de voyage. Le but était de leur créer un site web moderne pour qu'ils puissent montrer leurs offres et que les clients puissent faire des recherches de vols.

01
Création de la base de données

J'ai commencé par réfléchir à toutes les informations à stocker (voyages, prix, descriptions) sur un Cahier des charges. J'ai ensuite créé les tables sur MySQL pour que tout soit bien organisé.

02
Sécurité et connexion

J'ai utilisé le système de connexion de Laravel (Breeze) pour que les gens puissent se connecter. L'administrateur de l'agence a accès à un panneau de gestion (Dashboard) protégé que les clients ne peuvent pas voir.

03
Création du panneau d'administration

C'est l'interface pour l'agence. Elle permet d'ajouter un nouveau voyage, gérer des devis ou de supprimer une destination directement sur le site sans toucher au code.

04
Recherche de vols (API Amadeus)

J'ai connecté l'api afin d'afficher les vrais horaires et les prix des vols en direct pour les clients. Les infos viennent automatiquement de l'api.

05
Mise en place d'une politique de confidentialité

J'ai rédigé et mis en place une politique de confidentialité pour protéger les données des utilisateurs et respecter les réglementations du RGPD.

06
Espace client

Pour finir, j'ai fait la partie visible par les clients (consultation des offres, formulaire de contact)

Environnement technique

Laravel PHP MySQL Bootstrap API Amadeus

Conclusion

Ce très bon stage m'a permis de mettre mes connaissances en pratique sur un vrai projet. J'ai appris à être autonome en travaillant à distance et j'ai vraiment progressé sur le framework Laravel. Ça m'a montré comment on crée une application utile pour une entreprise.
